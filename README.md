# seerah-stories

Bismillah!

Generate seerah stories for kids based on Yasir Qadhi's seerah lectures.

You will need:

- YouTube API Key
- OpenAI API Key

Create a `.env` file with these:


```Dotenv
YOUTUBE_API_KEY="..."
OPENAI_API_KEY="..."
YOUTUBE_PLAYLIST_ID="PLAEA99D24CA2F9A8F"
```

Technical this should work for any YouTube playlist.
